# cypress-toggle-panel-plugin

## Install

```shell
npm install --save-dev cypress-toggle-panel-plugin
```

## Usage

```javascript
// cypress/support/e2e.js and/or cypress/support/component.js
import "cypress-toggle-panel-plugin/controls";
```
